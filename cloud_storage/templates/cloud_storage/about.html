About

임시 파일 클라우드는 로그인이 필요 없는 빠르고 단순한 파일 공유 서비스를 위해 제작되었습니다.

과제, 문서, 이미지 등 작은 파일을 빠르게 공유하고 싶을 때 사용하세요. 고유한 파일 번호와 4자리 비밀번호만으로 파일을 빠르게 업로드하고 다운로드할 수 있습니다.

주요 기능

  • 간편한 업로드: 파일과 4자리 비밀번호만으로 간편하게 업로드할 수 있습니다.
  • 고유 파일 번호: 업로드 시 생성된 고유 파일 번호를 통해 파일에 접근합니다.
  • 보안 접속: 파일 번호와 비밀번호를 모두 알아야만 파일을 다운로드할 수 있습니다.
  • 자동 삭제: 관리자가 설정한 기간이 지나면 파일이 자동으로 서버에서 삭제됩니다.
  • 주의! : 공개되어도 괜찮은 파일만 업로드해주세요. 사이트는 보안적 허점에 대해 보장할 수 없습니다.
